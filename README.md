# Samplist Feedback

This repository is the public feedback tracker for [Samplist](https://samplist.formantlabs.io/).

Samplist is a browser-based tool for building, generating, previewing, capturing, and exporting musical sequence ideas. It is being developed by Formant Labs as a practical workspace for MIDI/audio experimentation, harmony exploration, and sample-oriented creative workflows.

## Try Samplist

Open the app here:

[https://samplist.formantlabs.io/](https://samplist.formantlabs.io/)

## Send Feedback

Use GitHub Issues in this repository to report bugs, share feedback, or suggest improvements:

[Create a new issue](https://github.com/formantlabs-io/samplist-feedback/issues/new/choose)

Good feedback includes:

- What you were trying to do
- What happened
- What you expected instead
- Your browser and operating system
- Screenshots or recordings, if they help explain the issue

## Issue Types

- **Bug report**: something is broken or behaving unexpectedly.
- **Feedback**: a rough edge, confusing workflow, missing capability, or general reaction.

## Notes

This repository does not contain the Samplist application source code. It exists only to collect public feedback and bug reports.
